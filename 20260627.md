# 場としての大学図書館の再構築

# 概要
　学習支援は人的支援に依存し、生成AIによりその構造が再編される結果、図書館の「場」は物理空間ではなく相互作用の場（field）として再定義される。

## 1.1 これまでの経緯

　**旧** 大学設置基準では、
```
（図書等の資料及び図書館）
第三十八条　大学は、学部の種類、規模等に応じ、図書、学術雑誌、視聴覚資料その他の教育研究上必要な資料を、図書館を中心に系統的に備えるものとする。
２　図書館は、前項の資料の収集、整理及び提供を行うほか、情報の処理及び提供のシステムを整備して学術情報の提供に努めるとともに、前項の資料の提供に関し、他の大学の図書館等との協力に努めるものとする。
３　図書館には、その機能を十分に発揮させるために必要な専門的職員その他の専任の職員を置くものとする。
４　図書館には、大学の教育研究を促進できるような適当な規模の閲覧室、レフアレンス・ルーム、整理室、書庫等を備えるものとする。
５　前項の閲覧室には、学生の学習及び教員の教育研究のために十分な数の座席を備えるものとする。
```

# ２． 場の再定義
## 2.1 field として「場」を再定義
　そこで，前回軽く触れたとおり、大学図書館における「場」を物理的な空間（space/place）としてではなく，人・情報・制度の相互作用によって構成される関係的構造としての「場（field）」として再定義してみたいと思います。

　「デジタル・ライブラリー」について検討がなされた令和5年（2023）の時以上に、生成AIの普及とそれに伴う学習環境の変化は、大学図書館の戦略そのものに再設計を迫っているように思います。
さきに触れたように、令和4年大学設置基準の改正で大学図書館は、**電子化やＩＴ化を踏まえた規定に再整理** され、学習環境としての空間整備が不要になったわけではありませんが、
空間の位置づけは二次的なものとなった、と考えるのが自然です。

　他方で、学習あるいは学習支援（学習と学修とをめぐる議論はありますが、ここではとりあえず学習としています。）が成立するには、人的支援の課題、さらに、人とAIが協働する新しい学習支援というサービス設計そのものが、図書館機能の再定義の中心になってくるのではないでしょうか。

　このような変化は同時に、図書館という場所、さらに図書館という組織の境界を曖昧化させる、というよりは、図書館という場所や組織にこだわるだけではうまくいかないように思います。

　第一に、場所を択ばないオンラインを通じた学習支援はもはや補完的サービスではなく、第二に、学習支援センターやライティングセンターといった他の教育支援組織との連携、統合や再編がが不可欠です。その時、図書館は少なくとも従来型の独立した組織ではなく、大学全体の教育支援システムの中で他部門と連携しながら機能するネットワーク型の存在として、再定義が必要でしょう。

　以上の延長線上には、図書館が必ずしも固有の建物や物理的空間を前提としないという可能性もでてきます。学習支援の本質が空間ではなく（念のため、空間が不要ということではありません。言い換えれば、空間は条件の一つですが、絶対的に必要な条件ではないし、十分条件でもありません。）、人・AI・情報・制度の相互作用によって形成されるならば、必要なのはそれらの相互作用を維持し、適切に設計することです。すなわち、図書館の存立条件は「場所（space/place）」ではなく、学習と研究を成立させる関係的構造の維持と強化に求められることになる、というのが「場（field）」としての再定義の意味です。

　学習は、誰とどのように相互作用したかという関係構造に強く依存する、というのが2012年の中央教育審議会（中教審）の答申でいう「能動的学修」像であり、それは図書館を「場所」として捉えるだけでは理解できず、「相互作用の強度と配置」を扱う field の概念によって初めて説明可能になります。

## 2.2 fieldとは？
　ここで、fieldとは物理学的な電場とか磁場とか、重力場のアナロジーとして考えています。
　[wikipedia](https://ja.wikipedia.org/wiki/%E5%A0%B4)によれば、場とは以下の通り説明されています。

```
場（ば、英: field、工学分野では電界・磁界など界とも）とは、物理量を持つものの存在が、その近傍・周囲に連続的に影響を与えること、あるいはその影響を受けている状態にある空間のこと。
```

「場」としての図書館という場合の「場」が、電磁場・重力場をイメージさせるのは、

* 目に見えない
* 空間に広がる
* 相互作用を媒介する
* そこに存在する主体の振る舞いを変える

という点があげられます。
このイメージによって、「人が本質」ということを強調することができます。

　言い換えると、目に見える場所は必然ではなく、非来館でも学習支援自体は成立し、空間は媒介であって本質ではない、とも言えます。

## 2.3 fieldの構造
　図書館が、相互作用を媒介することによって、そこに存在する主体の振る舞いを変えるには、次の要素を考えることができます。

* 人的支援（第3章）
* 生成AI（第4章）

# ３．人的支援
　すでに述べたように、令和4年大学設置基準の改正で大学図書館は、場所ではなく人による支援をその機能とすることにシフトしています。場所ではなく、学習支援や研究支援といった機能を実際に担う人的支援こそが、図書館の実質的な価値を構成する中核的要素です。

　しかしながら、この支援機能は従来の図書館員の専門性と単純に重なりません。学習支援は、学習設計や対話的指導、さらには心理的支援といった要素を含む、より広範な能力を前提としており、既存の職能体系の延長線上でその役割を担うことには限界があり、支援機能と人材の不一致という構造的問題が生じています。

　このシフトを有効に行うためには、図書館（という場所）に **いる** 人材も再設計しないと成立しないことは明らかです。場所ではなく人による支援へのシフトをすることが、「場＝field」としての再定義の一番重要なポイントです。

# ４．生成AI
　もう一つ、相互作用を媒介し、そこに存在する主体の振る舞いを変えるためには、生成AIの存在を抜きには、これからの時代は成立しないでしょう。場所ではなく人による支援にシフトした、と書きましたが、支援の　**道具** としての生成AIを考えることが必要です。

# ５．まとめにかえて
　学習に場所が必要であること自体は疑いありません。図書館には閲覧席があり、これまで、明治時代に近代的な大学、大学図書館が始まって以来、伝統的に自習スペースとして活用されてきました。2012年の中央教育審議会答申で「能動的学修（アクティブ・ラーニング）」が推進されたことや、デジタル化の進展により蔵書利用が相対的に減少したことも契機となり、閲覧席の一部はラーニングコモンズへと転換されました。これが、いわば第一次の「場」としての図書館の再定義でした。

　ラーニングコモンズは、多くの大学で図書館主導により整備・運用され、一定の成果を上げています。しかし、学習支援の場として最適な機能を果たしているかという点では課題が残っています。管見の限りでは先進的な一部の例を除き、教育部門との連携が十分に進んでおらず、教育部門側においても、従来の「授業＋自習」という枠組みを超えた学習支援の仕組みが体系的に整備されているとは言い難い状況です。その結果、ラーニングコモンズは組織的な学習支援の中核として位置づけられるには至らず、限定的・付随的な機能にとどまっているのではないでしょうか。

　さらに、コロナ禍において物理的な場所の利用が制限された経験を経て、学習環境のあり方自体が見直されています。加えて、生成AIの急速な普及により、学習プロセスそのものが変化しつつあります。このような状況を踏まえると、「場」としての図書館の役割についても、改めて再検討が求められている段階にあります。いわば、ここがイマココ（現在地）です。

　繰り返しになりますが、学習に場所が必要である点は変わりませんが、その場所が図書館であることが必須なのか、あるいは最適解なのかについては、必ずしも十分に検証されておらず、未解決の論点と考えています。

　今後の方向性としては、大きく二つが想定されます。

　第一は、図書館が学習の場として最適解となるために、機能や運営の在り方を高度化していく方向です。たとえば、教育部門との連携強化や、学習プロセスに組み込まれた支援提供などにより、図書館を学習支援の中核に位置づけていくアプローチです。

　第二は、図書館がこれまで担ってきた自習スペース機能やラーニングコモンズ機能を、図書館という枠組みから切り離し、大学全体として最適な学習環境を再設計する方向です。極端に言えば、図書館の学習支援機能を一度解体・再配置し、全学的な視点で再構築する。その際、図書館が主導的にビジョンを提示し、全体最適を牽引する役割を担う、という考え方です。

　これに加えて、両者を折衷する第三の道、すなわち「図書館をハブとしつつ、機能は分散・連携させる」ようなモデルも検討余地があるかもしれません。その時に、図書館を単なる「場所（place/space）」ではなく、「場（field）」、すなわち多様な主体が相互作用を行うダイナミックな環境として再定義することが有効と考えられます。ここでいう「場（field）」とは、物理的な空間や蔵書にとどまらず、学生・教員・職員、さらにはデジタル資源や生成AIといった要素が交差し、学習・知識創造・支援が相互に作用し合う関係性の総体を指します。

　このように捉えると、図書館の価値は、誰と誰が出会い、どのような活動が誘発され、どのように学びが深化していくのかという「相互作用の設計」こそが中核的な役割となります。そのためには、物理空間の設計と同時に、プログラム設計、人的支援、さらにはデジタル環境との統合を一体的にマネジメントしていく必要があります。

　また、「場（field）」としての再定義は、図書館単体の最適化に閉じるものではありません。教育部門やICT部門、学習支援センター等との有機的な連携を通じて、学習プロセス全体の中に位置づけられることで、初めて実効性を持ちます。言い換えれば、図書館は「完結した場」ではなく、「学習エコシステムのハブ」として機能することが求められることになります。

　図書館は、固定的な機能の集積ではなく、相互作用を触媒するプラットフォームとして再編されるべき存在であり、その意味で改革の起点にもなり得る、という位置づけになります。本質は相互作用設計です。さしあたり、机上の議論でしかありませんが、実質化していく意味があるか、今後も考えてみたいと思います。
